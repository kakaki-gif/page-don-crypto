Don via NowPayments

Choisis la crypto et le montant en USD, puis clique sur faire un don.

Le widget va ouvrir une fenêtre NowPayments pour payer avec la crypto et le montant choisis. Les dons sont irréversibles. Pour usage personnel uniquement.

⚠️ Ne partage jamais tes clés privées. Ce widget utilise ta clé publique NowPayments dans le front ; pour une version plus sécurisée, il faudrait passer par un backend.

const amountInput = document.getElementById('amount'); const cryptoSelect = document.getElementById('crypto'); const widgetDiv = document.getElementById('nowpayments_widget'); function refreshWidget() { const amount = parseFloat(amountInput.value); const crypto = cryptoSelect.value.toUpperCase(); if (!amount || amount <= 0) { widgetDiv.removeAttribute('data-amount'); } else { widgetDiv.setAttribute('data-amount', amount.toFixed(2)); } widgetDiv.setAttribute('data-pay-currency', crypto); // Recrée le widget (hack pour forcer actualisation) const parent = widgetDiv.parentNode; const clone = widgetDiv.cloneNode(true); parent.replaceChild(clone, widgetDiv); } // Ecoute les changements amountInput.addEventListener('input', refreshWidget); cryptoSelect.addEventListener('change', refreshWidget); // Initialisation au chargement refreshWidget(); Mise à jour : widget NowPayments avec sélection montant/crypto
